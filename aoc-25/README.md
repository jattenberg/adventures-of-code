# Advent of Code 2025 - Haskell

Solutions to Advent of Code 2025 puzzles written in Haskell.

## Building

Using Stack:
```bash
stack build
stack exec aoc-25
```

Using Cabal:
```bash
cabal build
cabal run aoc-25
```

## Structure

- `src/` - Source code
- `resources/` - Input files (to be created)
